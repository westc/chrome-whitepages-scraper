Whitepages Scraper

Instructions

You need to select the four different files shown in the Files section and then click the Find People button to start searching.

Every once in a while Whitepages will attempt to prevent applications like this one from working by soliciting proof that there is a human carrying out these searches. At that time

Each of the files should be CSV files which may also be created and/or opened in a spreadsheet application such as Microsoft Excel. Here is what each file should contain.

  • Cities File
    This file should contain the zip codes to be searched. This file must contain at least one "Zip Code" column which must not be empty.
  • Last Names File
    This file should contain the last names to be searched. This file must contain at least one "Last Name" column which must not be empty.
  • Searches File
    This file will be an exhaustive list of all of the searches that should be carried out. The first time through, this can simply be a blank file if you'd like. Thereafter the file will contain the following columns: "Timestamp", "Zip Code", and "Last Name". The "Timestamp" will be a representation of when the search was actually completed.
  • Results File
    This will contain all of the search results gathered. It is the file which you can use to then compile your own list of phone numbers to be checked by a publisher and thereafter added to a phone territory. The first time through, this can simply be a blank file if you'd like. Once at least one matching record is found the file will contain the following columns: "Timestamp", "Zip Code", "Last Name", "Full Name", "Phone", "Address 1", "Address 2", "City", and "State".
